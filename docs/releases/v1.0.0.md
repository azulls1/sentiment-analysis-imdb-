## Descripcion General

Primera release completa del sistema de Analisis de Sentimientos para resenas de peliculas de IMDb. Incluye backend con FastAPI, frontend Angular, modelo de Machine Learning (SVM), integracion con Argilla para clasificacion zero-shot, suite completa de tests, infraestructura Docker y pipeline CI/CD.
