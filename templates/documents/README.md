# Document Templates - Salida de Skills

Esta carpeta es el destino para documentos Word (.docx) generados por los skills NXT.

## Templates Fuente

Los templates de documentos estan en:
- **Principales**: `plantillas/*.md` (NXT v3.6.0)
- **BMAD**: `nxt/method/templates/*.md` (referencia)

## Templates Disponibles

| Template | Ubicacion | Uso |
|----------|-----------|-----|
| Project Brief | `plantillas/project-brief.md` | Brief inicial |
| PRD | `plantillas/prd-template.md` | Documento de requisitos |
| Architecture | `plantillas/architecture-template.md` | Arquitectura |
| Story | `plantillas/story-template.md` | User stories |
| Test Plan | `plantillas/test-plan-template.md` | Plan de pruebas |

## Uso

El skill `SKILL-docx.md` en `skills/documentos/` usa estos templates
para generar documentos Word profesionales.

## Estilos NXT

Los documentos generados siguen el estilo NXT:
- Fuente: Calibri 11pt
- Titulos: Calibri Bold con jerarquia
- Margenes: 2.5cm
- Colores: #3B82F6 (azul), #F97316 (naranja)
