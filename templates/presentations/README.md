# Presentation Templates

Esta carpeta contiene templates para presentaciones PowerPoint (.pptx).

## Tipos de Presentaciones

### PRD Pitch (5-10 slides)
1. Título + problema
2. Solución propuesta
3. Usuarios objetivo
4. Features principales
5. Arquitectura de alto nivel
6. Roadmap
7. Próximos pasos
8. Q&A

### Sprint Demo (3-5 slides)
1. Objetivos del sprint
2. Lo completado (con screenshots)
3. Métricas y KPIs
4. Próximo sprint

### Kickoff de Proyecto (8-12 slides)
1. Visión del proyecto
2. Objetivos y KPIs
3. Equipo y roles
4. Timeline
5. Tech stack
6. Riesgos
7. Próximos pasos

## Estilo Visual NXT

- **Colores primarios**: #3B82F6 (azul), #F97316 (naranja)
- **Colores secundarios**: #10B981 (verde), #8B5CF6 (púrpura)
- **Fondo**: Blanco o gris claro (#F9FAFB)
- **Tipografía**: Sans-serif (Calibri, Arial)

## Uso

El skill `nxt-presentations` usa estos templates para generar
presentaciones profesionales.
