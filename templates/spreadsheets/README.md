# Spreadsheet Templates

Esta carpeta contiene templates para hojas de cálculo Excel (.xlsx).

## Templates Disponibles

### Backlog de Producto
Columnas: ID, Epic, Story Title, Description, Priority, Story Points, Sprint, Status, Assignee, Notes

### Sprint Planning
Columnas: Sprint #, Goal, Start Date, End Date, Capacity, Committed, Completed, Velocity, Status

### Bug Tracking
Columnas: Bug ID, Title, Severity, Related Story, Status, Found By, Fixed By, Date Found, Date Fixed

### Test Cases
Columnas: TC ID, Feature, Scenario, Steps, Expected Result, Actual Result, Status, Priority

### Risk Register
Columnas: Risk ID, Description, Category, Probability, Impact, Score, Mitigation, Owner, Status

## Formato Estándar

### Estilos
- **Header**: Fondo #3B82F6, texto blanco, negrita
- **Subtotal**: Fondo #E5E7EB, negrita
- **Total**: Fondo #1F2937, texto blanco
- **Alternating rows**: Fondo #F9FAFB

### Funcionalidades
- Filtros automáticos en headers
- Primera fila congelada
- Anchos de columna automáticos
- Validación de datos donde aplique
- Formato condicional para estados

## Uso

El skill `nxt-spreadsheets` usa estos templates para generar
spreadsheets profesionales.
